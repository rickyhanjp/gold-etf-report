## 📊 当前使用的数据源

### 主要数据源：AkShare

**AkShare** 是一个开源的 Python 财经数据接口库，提供全面的金融市场数据。

**官网**: https://akshare.akfamily.xyz  
**GitHub**: https://github.com/akfamily/akshare

---

### 具体接口

```python
# 使用的 AkShare 接口
ak.fund_etf_hist_em(
    symbol='510300',        # ETF 代码
    period='daily',         # 日线
    start_date='20160101',  # 开始日期
    end_date='20260329',    # 结束日期
    adjust='qfq'            # 前复权
)
```

**数据字段**:
- 日期 (date)
- 开盘 (open)
- 最高 (high)
- 最低 (low)
- 收盘 (close)
- 成交量 (volume)
- 成交额 (amount)

## 📊 数据质量

### 优点

1. **前复权处理** - 考虑分红、拆分
2. **数据连续** - 无缺失值
3. **权威来源** - 东方财富
4. **免费开放** - 无需 API Key

---

### 局限

1. **历史长度** - 部分 ETF 仅 8-10 年数据
2. **盘中数据** - 仅提供日线
3. **实时性** - 延迟 15 分钟

## ⚠️ 数据注意事项

### 1. 前复权处理

**使用 `adjust='qfq'` (前复权)**

**原因**:
- ✅ 考虑历史分红
- ✅ 考虑拆股
- ✅ 收益计算准确

---

### 2. 缺失值处理

```python
# 回测代码中处理
price_df = price_df.dropna()
```

**影响**:
- 部分 ETF 早期数据缺失
- 回测从数据完整日开始
