# 黄金趋势 + 因子策略 优化总结

## V1 vs V2 回测对比

| 指标 | V1 原版 | V2 优化版 | 改进 |
|------|--------|----------|------|
| 总收益率 | **+90.39%** | +44.64% | -45.75% ❌ |
| 年化收益 | **+17.47%** | +9.67% | -7.80% ❌ |
| 最大回撤 | 1.33% | **2.87%** | +1.54% ❌ |
| 夏普比率 | **9.17** | 6.05 | -3.12 ❌ |
| 平均仓位 | 86.67% | 100% | +13.33% ⚠️ |

## 问题分析

### V2 失败原因

1. **仓位过于激进**
   - V2 平均仓位 100%，完全失去风控
   - 在 2022 年市场调整时没有减仓

2. **趋势强度加成过度**
   - 强势多头 +25% 仓位导致始终满仓
   - 失去了策略的灵活性

3. **因子加权效果不佳**
   - 核心因子（TIPS、美元、ETF）加权 1.5x
   - 但在牛市中这些因子持续正向，导致始终满仓

4. **回测数据不足**
   - 仅 30 个月度数据点
   - 无法充分验证策略稳定性

## 正确的优化方向

### 1. 保持 V1 核心逻辑
- V1 的 3 档仓位（0%/50%/100%）表现良好
- 最大回撤控制优秀（1.33%）

### 2. 优化点应该是

#### A. 降低踏空风险
```python
# 原阈值
empty_threshold: 4    # 0-4 分空仓
half_threshold: 7     # 5-7 分半仓
full_threshold: 8     # 8-12 分满仓

# 优化阈值（更积极但不失风控）
empty_threshold: 3    # 0-3 分空仓
half_threshold: 6     # 4-6 分半仓
full_threshold: 8     # 7-12 分满仓
```

#### B. 添加趋势延续模块
- 当趋势连续 3 个月多头时，仓位下限提升至 50%
- 避免在强势趋势中过早空仓

#### C. 动态因子权重
- 牛市中提高动量因子权重
- 熊市中提高避险因子权重

#### D. 周度监控 + 月度调仓
- 每周监控趋势状态
- 月度执行调仓（平衡频率和成本）

### 3. 接入真实宏观数据

**必须接入的数据源：**

| 因子 | 数据源 | 频率 |
|------|--------|------|
| TIPS 实际利率 | 美联储 FRED | 每日 |
| 美元指数 | 彭博/路透 | 每日 |
| 黄金 ETF 持仓 | 世界黄金协会 | 每日 |
| COMEX 持仓 | CFTC COT | 周度 |
| 美国 CPI | 劳工部 | 月度 |
| 全球 PMI | 标普/ISM | 月度 |

### 4. 止损优化

**当前问题：**
- V1 无止损模块
- 完全依赖趋势反转退出

**建议添加：**
- 追踪止损：从高点回撤 8% 止损
- 时间止损：持仓 3 个月无收益减仓
- 趋势止损：跌破 MA50 快速平仓

## 推荐的 V2 优化方案

```python
# 仓位阈值微调

# 添加趋势延续
```

**预期效果：**
- 总收益提升 10-20%
- 回撤增加 0.5-1%
- 夏普比率基本持平

### 方案 2：激进优化（不推荐）

```python
# 更积极的仓位

half_threshold: 5     # 3-5 分半仓
```

**风险：**
- 回撤可能增加 2-3%
