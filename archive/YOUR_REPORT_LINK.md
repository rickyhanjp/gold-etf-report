   - 点击 **Commit changes**

3. **启用 Pages**
   - Settings → Pages
   - Source: Deploy from a branch
   - Branch: main → / (root)
   - Save

4. **访问报告**
   - 等待 1-2 分钟
   - 访问：`https://<你的用户名>.github.io/518880-backtest/`
