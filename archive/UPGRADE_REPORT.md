## 📈 性能指标说明

### 夏普比率 (Sharpe Ratio)
```
夏普比率 = (年化收益率 - 无风险利率) / 年化波动率
```
- > 1.0 : 优秀
- > 0.5 : 良好
- < 0 : 亏损

本回测中夏普比率 0.239，表现一般（市场波动大）。

### 最大回撤 (Max Drawdown)
```
最大回撤 = (峰值 - 谷值) / 峰值
```
表示历史上最大的亏损幅度。-28.23% 表示从最高点下跌了 28.23%。

### 胜率 (Win Rate)
```
胜率 = 盈利交易次数 / 总交易次数
```
网格策略胜率 61.54%，表示超过 6 成的交易是赚钱的。
