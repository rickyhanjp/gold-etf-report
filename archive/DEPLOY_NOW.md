## 📱 Netlify 部署（30 秒完成）

### 步骤 1：打开 Netlify Drop

点击链接打开：
**https://app.netlify.com/drop**

### 步骤 2：拖拽文件夹

将以下文件夹拖到 Netlify Drop 的上传区域：

```
C:\Users\Administrator\.openclaw\workspace\guojin_auto_trading\deploy_package
```

**或者** 直接拖拽 ZIP 文件：
```
C:\Users\Administrator\.openclaw\workspace\guojin_auto_trading\backtest_report_deploy.zip
```

### 步骤 3：等待上传完成

- 上传时间：约 10-30 秒
- 上传完成后会自动生成访问链接

### 步骤 4：获得在线链接

格式如下：
```
https://random-name-12345.netlify.app/report.html
```

## 🆘 遇到问题？

### 问题 1：拖拽没反应
**解决：** 使用 Chrome 或 Edge 浏览器

### 问题 2：上传失败
**解决：** 检查网络连接，或尝试 ZIP 文件上传

### 问题 3：页面空白
**解决：** 等待 1-2 分钟，Netlify 需要时间构建
