## 📊 策略核心公式

### 黄金公允价格计算

```
G_公允 = -25 × DXY - 40 × TY10 + 5445
```

**参数说明**:
- **DXY**: 美元指数（如 105.0）
- **TY10**: 10 年期美债收益率（如 4.25 表示 4.25%）
- **5445**: 基准常数（已包含信用溢价）

**系数含义**:
- 美元指数每涨 1 点 → 金价跌约 25 美元
- 10 年期美债收益率每涨 0.1% → 金价跌约 4 美元

## 🎯 估值判断规则

| 偏离率 | 估值状态 | 操作建议 | 建议仓位 |
|--------|---------|---------|---------|
| > +5% | 明显高估 | 强烈卖出 | 10% |
| +2% ~ +5% | 小幅高估 | 逢高减仓 | 30% |
| -2% ~ +2% | 合理估值 | 观望 | 50% |
| -5% ~ -2% | 小幅低估 | 逢低买入 | 70% |
| < -5% | 明显低估 | 分批买入 | 100% |

**偏离率计算**:
```
偏离率 = (当前金价 - G_公允) / G_公允 × 100%
```

## 🔧 与 V4 策略集成

### 替换 V4 的止盈止损模块

```python
from strategies.rsi_four_stage_v4 import RSIFourStageStrategyV4
from strategies.gold_pricing import GoldPricingStrategy, GoldValuationTrailingStop

# 1. 创建 V4 策略（禁用内置止盈止损）
v4_strategy = RSIFourStageStrategyV4({
    'use_trailing_profit': False,  # 禁用 V4 追踪止盈
    'use_stop_loss': False,         # 禁用 V4 固定止损
})

# 2. 创建黄金定价策略
gold_pricing = GoldPricingStrategy({
    'dxy_coefficient': -25,
    'ty10_coefficient': -40,
    'base_constant': 5445,
})

# 3. 创建基于估值的止盈止损
valuation_stop = GoldValuationTrailingStop(gold_pricing)

# 4. 集成使用
def analyze_with_gold_pricing(symbol, daily_klines, position, dxy, ty10):
    """
    使用黄金定价策略增强 V4 信号
    
    Args:
        symbol: 股票代码
        daily_klines: 日线数据
        position: 当前持仓
        dxy: 美元指数
        ty10: 10 年期美债收益率
    """
    # V4 策略分析
    v4_signal = v4_strategy.analyze(symbol, daily_klines, position)
    
    # 黄金定价分析
    gold_signal = gold_pricing.analyze(dxy, ty10, v4_signal.price)
    
    # 综合判断
    if gold_signal.valuation_state.name in ['STRONG_OVERVALUED', 'SLIGHTLY_OVERVALUED']:
        # 高估：提高卖出优先级
        if v4_signal.signal_type.name == 'SELL':
            v4_signal.confidence = min(0.95, v4_signal.confidence + 0.2)
            v4_signal.reason += f" + 金价高估{gold_signal.deviation_rate:.1f}%"
    
    elif gold_signal.valuation_state.name in ['STRONG_UNDERVALUED', 'SLIGHTLY_UNDERVALUED']:
        # 低估：提高买入优先级
        if v4_signal.signal_type.name == 'BUY':
            v4_signal.confidence = min(0.95, v4_signal.confidence + 0.2)
            v4_signal.reason += f" + 金价低估{abs(gold_signal.deviation_rate):.1f}%"
    
    # 动态止盈止损（基于估值）
    if position:
        stop_loss = valuation_stop.calculate_dynamic_stop_loss(
            v4_signal.price,
            gold_signal.fair_value,
            position['avg_cost'],
            gold_signal.valuation_state
        )
        take_profit = valuation_stop.calculate_dynamic_take_profit(
            v4_signal.price,
            gold_signal.fair_value,
            position['avg_cost'],
            gold_signal.valuation_state
        )
        
        v4_signal.stop_loss_price = stop_loss
        v4_signal.stop_profit_price = take_profit
    
    return v4_signal, gold_signal
```

## 📝 使用示例

### 基础用法

```python
from strategies.gold_pricing import GoldPricingStrategy

# 创建策略
strategy = GoldPricingStrategy()

# 分析当前市场
dxy = 105.0       # 美元指数
ty10 = 4.25       # 10 年期美债收益率
gold_price = 2650 # 当前金价

signal = strategy.analyze(dxy, ty10, gold_price)

print(f"公允价格：{signal.fair_value:.0f}美元")
print(f"偏离率：{signal.deviation_rate:+.1f}%")
print(f"估值状态：{signal.valuation_state.value}")
print(f"操作建议：{signal.action}")
print(f"建议仓位：{strategy.get_position_suggestion(signal.valuation_state)*100:.0f}%")
```

### 分批交易

```python
from strategies.gold_pricing import GoldPricingStrategy, GoldValuationTrailingStop

pricing = GoldPricingStrategy()
trailing_stop = GoldValuationTrailingStop(pricing)

fair_value = 2600
current_price = 2450
position = 0.5  # 当前 5 成仓

# 检查买入信号
buy_signal, buy_ratio, reason = trailing_stop.check_buy_signal(
    current_price, fair_value, position
)

if buy_signal:
    print(f"触发买入：{reason}")
    print(f"买入比例：{buy_ratio*100:.0f}%")

# 检查卖出信号
sell_signal, sell_ratio, reason = trailing_stop.check_sell_signal(
    current_price, fair_value, position
)

if sell_signal:
    print(f"触发卖出：{reason}")
    print(f"卖出比例：{sell_ratio*100:.0f}%")
```

### 实时数据集成

```python
# 从 API 获取实时数据
import requests

# 1. 获取美元指数
dxy_response = requests.get('https://api.example.com/dxy')
dxy = dxy_response.json()['price']

# 2. 获取 10 年期美债收益率
ty10_response = requests.get('https://api.example.com/ty10')
ty10 = ty10_response.json()['yield']

# 3. 获取黄金价格
gold_response = requests.get('https://api.example.com/gold')
gold_price = gold_response.json()['price']

# 4. 分析
strategy = GoldPricingStrategy()
signal = strategy.analyze(dxy, ty10, gold_price)

# 5. 执行交易
if signal.action == "分批买入":
    position = strategy.get_position_suggestion(signal.valuation_state)
    execute_buy(position)
elif signal.action == "分批卖出":
    position = strategy.get_position_suggestion(signal.valuation_state)
    execute_sell(1 - position)
```

## 📊 实战案例

### 案例 1：2024 年 3 月（美元走强）

```
输入:
- DXY = 108.0
- TY10 = 4.50%
- 金价 = 2650

计算:
G_公允 = -25×108 - 40×4.50 + 5445
       = -2700 - 180 + 5445
       = 2565 美元

偏离率 = (2650 - 2565) / 2565 × 100% = +3.3%

判断：小幅高估
建议：逢高减仓
仓位：30%
```

### 案例 2：2024 年 6 月（美元走弱）

```
输入:
- DXY = 100.0
- TY10 = 3.50%
- 金价 = 2650

计算:
G_公允 = -25×100 - 40×3.50 + 5445
       = -2500 - 140 + 5445
       = 2805 美元

偏离率 = (2650 - 2805) / 2805 × 100% = -5.5%

判断：明显低估
建议：分批买入
仓位：100%
```

## ⚠️ 注意事项

### 1. 数据更新频率

| 数据 | 更新频率 | 来源 |
|------|---------|------|
| DXY | 实时 | 美联储/彭博 |
| TY10 | 实时 | 美国财政部/彭博 |
| 金价 | 实时 | LBMA/COMEX |

**建议**: 每日更新一次，短期交易可实时更新

### 2. 模型局限性

- ✅ 适合短期（1-4 周）判断
- ⚠️ 不适合长期预测
- ⚠️ 极端行情下可能失效
- ⚠️ 地缘政治事件未纳入

### 3. 参数调整

如遇模型偏离，可调整基准常数：

```python
# 重新拟合公式
# G_公允 = -25 × DXY - 40 × TY10 + X
# X = 当前金价 + 25×DXY + 40×TY10

current_gold = 2700
dxy = 105
ty10 = 4.25

new_constant = current_gold + 25*dxy + 40*ty10
# new_constant = 2700 + 2625 + 170 = 5495

strategy = GoldPricingStrategy({
    'base_constant': new_constant,
})
```

- ✅ 黄金 ETF（518880）
