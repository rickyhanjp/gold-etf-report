## 🎯 V4 策略 6 大核心功能

### 1. ADX 趋势强度过滤

```
ADX > 25: 强趋势市 → 顺势交易
ADX < 20: 震荡市 → 高抛低吸
ADX 20-25: 过渡区 → 观望
```

**作用**: 区分市场类型，调整策略

---

### 2. 多时间周期分析

```
日线 RSI: 主信号
周线 RSI: 趋势确认

买入条件:
- 日线 RSI < 30
- 周线 RSI < 50（确认）

卖出条件:
- 日线 RSI > 70
- 周线 RSI > 50（确认）
```

**作用**: 避免日线假信号

---

### 3. 智能仓位管理

```python
仓位 = 基础仓位 (50%)
     + 置信度加成 (最高 +20%)
     + ADX 趋势加成 (最高 +10%)
     + 背离加成 (最高 +15%)
     + 市场状态加成 (最高 +10%)
     - 风险减成 (最高 -20%)

范围：20% - 100%
```

**作用**: 高置信度重仓，低置信度轻仓

---

### 4. 分级止盈

```
盈利 5%:  卖出 30% → 锁定部分利润
盈利 10%: 卖出 30% → 继续锁定
盈利 15%: 卖出 40% → 清仓

追踪止盈：最高价回撤 5% 触发
```

**作用**: 既锁定利润，又让利润奔跑

---

### 5. 市场状态识别

```
牛市：价格>MA20>MA60>MA120
  → RSI 阈值 25/75（更宽松）
  → 积极做多

熊市：价格<MA20<MA60<MA120
  → RSI 阈值 35/65（更严格）
  → 轻仓操作

震荡市：均线纠缠
  → RSI 阈值 30/70（默认）
  → 高抛低吸
```

**作用**: 动态调整策略参数

---

### 6. 止损保护

```
固定止损：8%
触发后强制卖出

作用：控制最大亏损
```

## 🔧 V4 策略使用指南

### 快速开始

```python
from strategies.rsi_four_stage_v4 import RSIFourStageStrategyV4

# 创建策略（启用所有功能）
strategy = RSIFourStageStrategyV4({
    'use_ma55_filter': True,
    'use_macd_confirm': True,
    'use_adx_filter': True,
    'use_multi_timeframe': True,
    'use_smart_position': True,
    'use_trailing_profit': True,
    'use_stop_loss': True,
    'use_dynamic_threshold': True,
})

# 准备数据
daily_klines = get_klines('518880.SH', '1d', 60)
weekly_klines = get_klines('518880.SH', '1w', 20)

# 当前持仓
position = {
    'symbol': '518880.SH',
    'quantity': 1000,
    'avg_cost': 4.80,
}

# 分析信号
signal = strategy.analyze('518880.SH', daily_klines, position, weekly_klines)

# 输出结果
print(f"信号：{signal.signal_type.value}")
print(f"建议仓位：{signal.suggested_position*100:.0f}%")
print(f"止盈价：{signal.stop_profit_price}")
print(f"止损价：{signal.stop_loss_price}")
```
