# 黄金定价策略 V2 使用指南

**版本**: V2.0.0（含美债信用 CDS5）  
**完成时间**: 2026-03-21  
**适用**: 短期（1-4 周）黄金交易决策

---

## 📊 策略核心公式

### 黄金公允价格计算

```
G_公允 = -25 × DXY - 40 × TY10 + 3500 + 1.5 × (CDS5 - 25)
```

**参数说明**:
- **DXY**: 美元指数（如 105.0）
- **TY10**: 10 年期美债收益率（如 4.25 表示 4.25%）
- **CDS5**: 美国 5 年期主权 CDS（bp，如 30 表示 30bp）
- **3500**: 基准常数（含央行购金、通胀中枢）
- **1.5 × (CDS5 - 25)**: 美债信用风险溢价（基准 25bp）

**系数含义**:
- 美元指数每涨 1 点 → 金价跌约 25 美元
- 10 年期美债收益率每涨 0.1% → 金价跌约 4 美元
- CDS5 每高 1bp → 金价涨约 1.5 美元

---

## ⚠️ 重要说明：公式校准

### 当前公式局限性

根据 2026 年实际金价（约 2650-2700 美元），原公式计算的公允价格偏低：

```
示例：DXY=105, TY10=4.25%, CDS5=30bp

G_公允 = -25×105 - 40×4.25 + 3500 + 1.5×(30-25)
       = -2625 - 170 + 3500 + 7.5
       = 712.5 美元

但实际金价约 2650 美元，偏离率 = (2650-712.5)/712.5 = +272%
```

### 原因分析

当前黄金已不是传统利率定价资产，而是：
- **信用对冲资产**（对冲美债信用风险）
- **货币贬值对冲**（对冲全球货币超发）
- **地缘政治对冲**（对冲战争/制裁风险）

这些因素未完全纳入公式。

### 解决方案

**方案 1：调整基准常数**

```
G_公允 = -25 × DXY - 40 × TY10 + 5445 + 1.5 × (CDS5 - 25)

测试：DXY=105, TY10=4.25%, CDS5=30
G_公允 = -2625 - 170 + 5445 + 7.5 = 2657.5 美元 ✓
```

**方案 2：使用相对估值**

不关注绝对公允价格，而关注**偏离率的变化趋势**：
- 偏离率上升 → 金价相对美元/美债走强
- 偏离率下降 → 金价相对美元/美债走弱

**方案 3：结合 V4 策略**

将黄金定价作为**辅助指标**，与 RSI 四阶段策略配合使用。

## 🎯 估值判断规则

| 偏离率 | 估值状态 | 操作建议 | 建议仓位 |
|--------|---------|---------|---------|
| > +6% | 严重高估 | 强烈卖出 | 10% |
| +2% ~ +6% | 小幅高估 | 逢高减仓 | 30% |
| -2% ~ +2% | 合理估值 | 观望 | 50% |
| -6% ~ -2% | 小幅低估 | 逢低买入 | 70% |
| < -6% | 严重低估 | 分批买入 | 100% |

**偏离率计算**:
```
偏离率 = (当前金价 - G_公允) / G_公允 × 100%
```

## 🔧 与 V4 策略集成

### 替换 V4 的止盈止损模块

```python
from strategies.rsi_four_stage_v4 import RSIFourStageStrategyV4
from strategies.gold_pricing import GoldPricingStrategy, GoldValuationTrailingStop

# 1. 创建 V4 策略（禁用内置止盈止损）
v4_strategy = RSIFourStageStrategyV4({
    'use_trailing_profit': False,
    'use_stop_loss': False,
})

# 2. 创建黄金定价策略
gold_pricing = GoldPricingStrategy({
    'dxy_coefficient': -25,
    'ty10_coefficient': -40,
    'base_constant': 5445,  # 调整至匹配当前金价
    'cds5_coefficient': 1.5,
    'cds5_baseline': 25,
})

# 3. 创建基于估值的止盈止损
valuation_stop = GoldValuationTrailingStop(gold_pricing)

# 4. 集成使用
def analyze_with_gold_pricing(symbol, daily_klines, position, dxy, ty10, cds5):
    """使用黄金定价策略增强 V4 信号"""
    
    # V4 策略分析
    v4_signal = v4_strategy.analyze(symbol, daily_klines, position)
    
    # 黄金定价分析
    gold_signal = gold_pricing.analyze(dxy, ty10, cds5, v4_signal.price)
    
    # 综合判断
    if gold_signal.valuation_state.name in ['STRONG_OVERVALUED', 'SLIGHTLY_OVERVALUED']:
        # 高估：提高卖出优先级
        if v4_signal.signal_type.name == 'SELL':
            v4_signal.confidence = min(0.95, v4_signal.confidence + 0.2)
            v4_signal.reason += f" + 金价高估{gold_signal.deviation_rate:.1f}%"
    
    elif gold_signal.valuation_state.name in ['STRONG_UNDERVALUED', 'SLIGHTLY_UNDERVALUED']:
        # 低估：提高买入优先级
        if v4_signal.signal_type.name == 'BUY':
            v4_signal.confidence = min(0.95, v4_signal.confidence + 0.2)
            v4_signal.reason += f" + 金价低估{abs(gold_signal.deviation_rate):.1f}%"
    
    # 动态止盈止损（基于估值）
    if position:
        stop_loss = valuation_stop.calculate_dynamic_stop_loss(
            v4_signal.price,
            gold_signal.fair_value,
            position['avg_cost'],
            gold_signal.valuation_state
        )
        take_profit = valuation_stop.calculate_dynamic_take_profit(
            v4_signal.price,
            gold_signal.fair_value,
            position['avg_cost'],
            gold_signal.valuation_state
        )
        
        v4_signal.stop_loss_price = stop_loss
        v4_signal.stop_profit_price = take_profit
    
    return v4_signal, gold_signal
```

## 📝 使用示例

### 基础用法

```python
from strategies.gold_pricing import GoldPricingStrategy

# 创建策略（使用校准后的参数）
strategy = GoldPricingStrategy({
    'base_constant': 5445,  # 调整至匹配当前金价
})

# 分析当前市场
dxy = 105.0       # 美元指数
ty10 = 4.25       # 10 年期美债收益率
cds5 = 30         # 5 年期 CDS
gold_price = 2650 # 当前金价

signal = strategy.analyze(dxy, ty10, cds5, gold_price)

print(f"公允价格：{signal.fair_value:.0f}美元")
print(f"偏离率：{signal.deviation_rate:+.1f}%")
print(f"估值状态：{signal.valuation_state.value}")
print(f"操作建议：{signal.action}")
print(f"信用溢价：{signal.credit_premium:+.0f}美元")
```

### 实时数据集成

```python
# 从 API 获取实时数据
import requests

# 1. 获取美元指数
dxy_response = requests.get('https://api.example.com/dxy')
dxy = dxy_response.json()['price']

# 2. 获取 10 年期美债收益率
ty10_response = requests.get('https://api.example.com/ty10')
ty10 = ty10_response.json()['yield']

# 3. 获取 5 年期 CDS
cds5_response = requests.get('https://api.example.com/cds5')
cds5 = cds5_response.json()['spread']

# 4. 获取黄金价格
gold_response = requests.get('https://api.example.com/gold')
gold_price = gold_response.json()['price']

# 5. 分析
strategy = GoldPricingStrategy({'base_constant': 5445})
signal = strategy.analyze(dxy, ty10, cds5, gold_price)

# 6. 执行交易
if signal.action == "分批买入":
    position = strategy.get_position_suggestion(signal.valuation_state)
    execute_buy(position)
elif signal.action == "分批卖出":
    position = strategy.get_position_suggestion(signal.valuation_state)
    execute_sell(1 - position)
```

## ⚠️ 注意事项

### 1. 数据更新频率

| 数据 | 更新频率 | 来源 |
|------|---------|------|
| DXY | 实时 | 美联储/彭博 |
| TY10 | 实时 | 美国财政部/彭博 |
| CDS5 | 实时 | Markit/彭博 |
| 金价 | 实时 | LBMA/COMEX |

**建议**: 每日更新一次，短期交易可实时更新

### 2. 模型局限性

- ✅ 适合短期（1-4 周）判断
- ⚠️ 不适合长期预测
- ⚠️ 极端行情下可能失效
- ⚠️ 地缘政治事件未完全纳入

### 3. 参数校准

**基准常数校准公式**:

```
base_constant = 当前金价 + 25×DXY + 40×TY10 - 1.5×(CDS5-25)

示例：金价=2650, DXY=105, TY10=4.25, CDS5=30
base_constant = 2650 + 2625 + 170 - 7.5 = 5437.5 ≈ 5445
```

## 📝 总结

### V2 策略优势

✅ **三因子模型**: DXY + TY10 + CDS5  
✅ **信用溢价**: 纳入美债信用风险  
✅ **分批交易**: 高估分批卖，低估分批买  
✅ **动态风控**: 基于估值调整止盈止损

### 使用建议

1. **校准基准常数**: 根据当前金价调整 base_constant
2. **关注偏离率趋势**: 比绝对值更重要
3. **结合 V4 策略**: 与 RSI 四阶段配合使用
4. **实时监控 CDS5**: 信用风险变化是重要信号
