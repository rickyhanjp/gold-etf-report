1. TIPS 实际利率（下行→+1）

4. 央行购金（净购金→+1）
5. COMEX 持仓（非超买→+1）
6. VIX（上升→+1）
7. 原油/黄金比价（上行→+1）
8. 美欧利差（收窄→+1）
9. 财政赤字（扩大→+1）
10. PMI（弱于预期→+1）
11. ETF 持仓（环比增加→+1）
12. 美债期限利差（倒挂→+1）

**仓位规则**:
```python
if macro_score <= 4:
    monthly_position = 0.0  # 空仓
elif macro_score <= 7:
    monthly_position = 0.5  # 半仓
else:
    monthly_position = 1.0  # 满仓
```

---

### 日线级别：乖离率 + 黄金分割

**乖离率计算**:
```python
bias_20 = (close - ma20) / ma20 * 100
```

**操作规则**:
```python
if bias_20 > 25:
    daily_position = 0.5  # 清仓浮动仓
elif bias_20 > 15:
    daily_position = 0.7  # 部分止盈
elif bias_20 < -5:
    daily_position = 0.9  # 负乖离买入
```

**黄金分割计算**:
```python
diff = high_60 - low_60
fib_ratio = (high_60 - current_price) / diff

if fib_ratio >= 0.618:
    daily_position = 1.0  # 极端回调，重仓
elif fib_ratio >= 0.500:
    daily_position = 0.9  # 深度回调，加仓
elif fib_ratio >= 0.382:
    daily_position = 0.8  # 中度回调，买回
```
